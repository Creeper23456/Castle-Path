# 城堡幽径 幻想故事集

## 前言

这是一篇写给自己的故事集，写给属于自己幻想的故事集。

故事发生在一个幻想世界中。主人公托马斯为了寻得世界的真相，开始了自己的学习之旅。

一路上他遇到了许多乐意帮助他的人，并在最终遇见了自己的挚爱。随后，一位神秘女子出现，打破了他对世界的一切认知。

梦醒时分，他发现整个世界，仅仅是自己在一次午睡中做过的梦。

也许结局并非重要，珍贵之处在于旅途的点点滴滴吧。

本故事集依旧处于WIP阶段，会随着个人闲暇时继续在Github发布新的后续。

灵感来源于《爱丽丝梦游仙境》。

## 与本创作集的关系

故事集是对曲目的补充。每一首单曲都可以在稍作改编后作为场景音乐使用。

希望各位听众在享受音乐的同时，也能读读背后的故事吧。
